How to Setup Blog with Hexo

Oct 08, 2015

date: 2015-10-08 20:58:43

This article is the introduction of how to setup your own blog on github.
Github could serve the content inside .github.com on the domain .github.io. Here we’re using hexo(Node.js) to generate static web pages and push them onto repo. In the other words, you could also use(or create)other blog framework to do this job.

Preparations

Register on Github

You could find Github here. BTW, choose a good username otherwise you will be regret later.

Install GIT

You could download GIT here, for windows users, you might need to set GIT to your system path or use the git shell.

Install Node.js

You could download Node.js here, just use the binary version is ok:).

Install Hexo

$ npm install hexo-cli -g

Setup blog

  1. Goto a path you choose to place your blog’s source code later
  2. $ hexo init <folder>
  3. $ cd <folder>
  4. $ npm install
  5. Goto Github, create a repo named as “.github.com”, yes you must use this name.
  6. Open the _config.yml file which created by the step 4.
  7. Scroll down to the bottom and edit the file like this:
  8. $ hexo clean
  9. $ hexo generate
  10. $ hexo deploy

Now you could goto .github.io to check your blog!

You could also use “hexo server” to run the blog on localhost before deploy to the real site:)
